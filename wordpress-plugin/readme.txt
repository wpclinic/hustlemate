WordPress monitoring and management plugin for WP Wingman platform.

== Description ==

WP Wingman Client connects your WordPress site to the WP Wingman platform for comprehensive monitoring, management, and support.

**Features:**

* Real-time uptime monitoring
* Performance metrics tracking
* Security scanning
* Automated backups
* Update management
* Remote site management
* Detailed reporting

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wp-wingman-client/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WP Wingman settings and enter your API key
4. Configure monitoring and backup options

== Frequently Asked Questions ==

= Where do I get an API key? =

Sign up at https://wpwingman.com to get your API key.

= Is this plugin free? =

The plugin is free, but requires a WP Wingman account.
